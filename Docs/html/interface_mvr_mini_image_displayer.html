Detailed Description

This displayer shows images on the table as thumbnails; this is similar to how images are shown in the Mover application.

This displayer is available as part of Mover Mini but is not registered by default. If you want to display images this way, you can register an instance of this class by calling its addDisplayer (MvrMiniImageDisplayer) method.
