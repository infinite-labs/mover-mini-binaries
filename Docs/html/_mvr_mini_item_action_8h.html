enum MvrMiniItemActionAfterPerforming

This enumeration includes actions the table can perform on your behalf after an MvrMiniItemAction is executed for a particular item.

To have these actions executed, set the MvrMiniItemAction::actionAfterPerforming property, or see the uses of this type in performActionOnItem: (MvrMiniItemAction) and MvrMiniItemActionBlock.

Enumerator:
kMvrMiniItemActionDoNothing 

Does nothing after the action is performed.

For actions that use blocks as their implementation, this is the default.

kMvrMiniItemActionHideTableAfterwards 

Hides the table after the action is performed.
