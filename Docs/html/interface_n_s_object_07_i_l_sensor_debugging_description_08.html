- (id) descriptionForDebugging  

If implemented, ILSensorSink will use the return value of this method as the JSON fragment describing this object.

It can be a NSString, NSNumber, NSNull object, or a NSArray of such objects, or a NSDictionary of NSString keys and any such objects.
